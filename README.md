# Civic Issues Frontend – Issue Management Dashboard

A modern React-based frontend dashboard for the Civic Issues Backend, built to help students and administrators report, track, and manage civic issues in college hostels and institutes.

This application consumes the FastAPI backend and provides a clean, responsive user interface for issue reporting, voting, commenting, and administrative management.

---

## Features

### Student Features
- Report new civic issues
- View all reported issues
- Upvote and downvote issues
- Comment on issues for discussion and updates
- Track issues reported by the logged-in user
- View real-time issue status updates  
  (new → acknowledged → in_progress → resolved → closed)

### Admin Features
- View and manage all reported issues
- Update issue status and priority
- Assign issues to administrators or staff
- Add resolution notes
- View analytics and statistics dashboards
- Filter and sort issues by status, category, and building
